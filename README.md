## Goal

Train AI to play simplified implementation of Tetris

## Game setup

- 20 X 10 well
- 5 actions (fixed) possible before the tetramino sinks 1 row
- next tetramino to appear is selected using random.randint()

## State space

Huge, 2<sup>200</sup> states

## Action space

- Move left
- Move right
- Rotate left
- Rotate right
- Do nothing
